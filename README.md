# gotmpl

A CLI for golang template.

## Usage

```
❯ cat text.txt
I'm {{ exec "whoami" | trim }}

❯ gotmpl text.txt
I'm skaji
```

## Available funcsions

* https://masterminds.github.io/sprig/
* exec
* fromYaml
* fromYamlMulti
* readFile
* toYaml
